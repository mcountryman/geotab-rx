Driver

Driver: = "Driver"
  • Defined in packages/mygeotab/src/models.ts:1912

User-based Driver. Makes a IDriver until a different driver id is assigned to the same device. First record in a trip overrides any other record encountered during the trip it was added to, except "TripDriver".

DriverKey

DriverKey: = "DriverKey"
  • Defined in packages/mygeotab/src/models.ts:1899

Driver Key - based Driver. Drivers use this key type to identify themselves. This will typically be used when the driver ID relay is enabled. Makes a IDriver until a different driver ID is assigned to the same device.
