Interface IDVIRLogSearch

The object used to specify the arguments when searching for IDVIRLog(s). A trailerSearch and deviceSearch cannot be used at the same time because a DVIR log entry is only ever associated with one asset type (for instance, if the "device" is set, "trailer" is always null and vice versa).
